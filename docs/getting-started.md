# Getting Started

## What exists today

`baseliner` currently provides a phase-0 CLI skeleton:

- `baseliner --version`
- `baseliner scan` (placeholder output only)

## Prerequisites

- Python `>=3.12`
- `uv`

## Setup

```bash
git clone <your-repo-url>
cd baseliner
uv sync --all-extras
```

## Run

```bash
uv run baseliner --version
uv run baseliner scan
```

Expected output for `scan` right now:

```text
not yet implemented
```
