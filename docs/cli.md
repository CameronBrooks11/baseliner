# CLI Reference

## Root command

```bash
uv run baseliner --help
```

Current options:

- `--version` show version and exit
- `--help` show help

Current command groups:

- `scan`

## `scan`

```bash
uv run baseliner scan --help
```

Options:

- `--config TEXT` path to config file (default: `baseliner.yaml`)
- `--help` show help

Current behavior:

```bash
uv run baseliner scan
```

Prints:

```text
not yet implemented
```
